# Xbox Audio Files Required

Place the following audio files in this directory for the Xbox/Halo audio experience:

## Boot Animation Sounds
| File | Description | Source Reference |
|------|-------------|------------------|
| `boot-ambient.mp3` | Halo 3 loading screen ambient drone | [YouTube Reference](https://www.youtube.com/watch?v=0ZHBJkfU1w0) |
| `boot-whoosh.mp3` | Transition whoosh for flash effect | Original Xbox boot |

## Menu Navigation Sounds (Xbox Dashboard)

| `menu-back.mp3` | Back/cancel sound | Original Xbox Dashboard |

## General UI Sounds
| File | Description |

## Where to Get These Sounds

### Option 1: Extract from YouTube Videos
1. Use a YouTube to MP3 converter
2. Trim to the specific sound effect you need
3. Export as MP3 (128-192kbps recommended)

### Option 2: Royalty-Free Alternatives
- [Freesound.org](https://freesound.org) - Search for "sci-fi UI", "menu blip", "whoosh"
- [Mixkit](https://mixkit.co/free-sound-effects/) - Free UI sound effects
- [ZapSplat](https://www.zapsplat.com/) - Sci-fi and UI sounds

### Option 3: Create Your Own
- Use a synthesizer to create retro-futuristic blips
- The classic Xbox hover sound is a short sine wave with pitch bend

## Recommended Audio Specs
- Format: MP3
- Sample Rate: 44100 Hz
- Bit Rate: 128-192 kbps
- Keep files under 100KB for quick loading
